# Tableau Dashboards

This repository contains **interactive Tableau dashboards** built to practice and demonstrate
**data visualization, analytical thinking, and business storytelling**.

All dashboards focus on turning raw data into **clear, actionable insights**.

---

##  Sales Performance Dashboard

- Total Sales, Profit & Quantity trends  
- Category & Region-wise performance  
- Monthly and yearly comparisons  
- Top & Bottom performing products  

**Features:**
- Interactive filters
- Parameters
- Highlight actions

**Skills Used:**  
`Tableau Desktop` 路 `Calculated Fields` 路 `Parameters` 路 `Dashboard Actions`

---

##  Business / Customer Insights Dashboard

- Customer segmentation  
- Revenue contribution analysis  
- Trend identification  

**Skills Used:**  
`LOD Expressions` 路 `Advanced Filters` 路 `Data Blending`
